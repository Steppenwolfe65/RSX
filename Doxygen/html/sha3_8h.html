void sha3_finalize ( uint64_t *  state,
size_t  rate,
const uint8_t *  message,
size_t  msgoffset,
size_t  msglen,
uint8_t *  output 
)

Finalize the message state and returns the hash value in output. Absorb the last block of message and create the hash value.
Produces a 32 byte output code using SHA3_256_RATE, 64 bytes with SHA3_512_RATE.

Warning
The output array must be sized correctly corresponding to the absorbtion rate ((200 - rate) / 2).
Finalizes the message state, can not be used in consecutive calls.
Parameters
stateThe function state; must be initialized
rateThe rate of absorption, in bytes
messageThe input message byte array
msglenThe number of message bytes to process
msgoffsetThe current position within the input message byte array
outputThe output byte array; receives the hash code
